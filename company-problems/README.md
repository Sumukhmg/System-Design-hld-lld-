## How to Practice
1. Pick a company → pick a problem.
2. Set a 45-min timer; solve it using the [7-step framework](../fundamentals/09-interview-framework.md).
3. Compare with the linked case study / building blocks.
4. Note the gaps; redo the deep-dive section.

## General prep advice by level
- **SDE-1 / new grad:** focus on **LLD** (parking lot, vending machine) + fundamentals + estimation.
- **SDE-2:** both **LLD** and **HLD** (URL shortener, chat, Twitter).
- **Senior / Staff:** **HLD** depth — trade-offs, bottlenecks, multi-region, consistency, the [key-value store](../hld/case-studies/key-value-store.md).
